# NFT Preview Card Component

## Descripción General

NFT Preview Card Component es una interfaz de usuario responsive desarrollada como parte de un reto de Frontend Mentor. El proyecto se centra en reproducir un diseño profesional utilizando HTML semántico y CSS moderno, construyendo un componente reutilizable que podría formar parte de aplicaciones web más grandes.

Los retos de Frontend Mentor están diseñados para mejorar habilidades de desarrollo frontend mediante la recreación de interfaces de calidad profesional a partir de especificaciones de diseño reales.

---

## Motivación

Los componentes pequeños de interfaz son excelentes ejercicios para fortalecer los fundamentos de la ingeniería frontend.

Este proyecto fue desarrollado para practicar:

- HTML semántico.
- Maquetación moderna con CSS.
- Diseño responsive.
- Interacciones hover.
- Componentes reutilizables.
- Implementación pixel-perfect.

Aunque visualmente sencillo, este tipo de tarjetas aparece constantemente en aplicaciones modernas y requiere especial atención a los espacios, la tipografía y la accesibilidad.

# Preguntas Frecuentes

## ¿Qué es este proyecto?

Es un componente responsive de vista previa NFT desarrollado como un reto de Frontend Mentor para practicar fundamentos del desarrollo frontend.

## ¿Qué tecnologías utiliza?

El proyecto fue construido completamente con HTML y CSS, sin frameworks externos ni JavaScript.

## ¿Cuál era el objetivo principal de aprendizaje?

Mejorar habilidades relacionadas con diseño responsive, HTML semántico, componentes reutilizables e implementación precisa de diseños profesionales.

## ¿Por qué desarrollar un componente tan pequeño?

Las aplicaciones web modernas están compuestas por componentes reutilizables. Aprender a construirlos correctamente establece bases sólidas para proyectos frontend más grandes.

## ¿Para quién está pensado este proyecto?

Para desarrolladores interesados en desarrollo web responsive, arquitectura CSS y diseño de interfaces reutilizables.
