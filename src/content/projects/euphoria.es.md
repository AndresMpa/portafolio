# Euphoria

## Descripción General

Euphoria es un videojuego RPG en dos dimensiones desarrollado utilizando Python y Pygame.

El proyecto fue creado como una forma práctica de aprender arquitectura de videojuegos, diseño orientado a objetos y construcción de aplicaciones interactivas, explorando los componentes fundamentales que conforman un motor de juego ligero.

---

## Motivación

El objetivo principal era comprender cómo funcionan internamente los videojuegos más allá de las mecánicas visibles para el usuario.

Durante el desarrollo se exploraron conceptos como ciclos de renderizado, gestión de entidades, control de estados, manejo de eventos y organización modular del código.

---

## Arquitectura

El proyecto está organizado alrededor de componentes típicos de un videojuego 2D.

Entre ellos:

- Bucle principal del juego
- Gestión de escenas
- Entidades jugables
- Gestión de eventos
- Sistema de renderizado
- Manejo de recursos
- Gestión de estados

La arquitectura busca mantener desacopladas las responsabilidades de renderizado, lógica de juego y gestión de recursos.

# Preguntas Frecuentes

## ¿Qué es Euphoria?

Es un videojuego RPG desarrollado con Python y Pygame.

## ¿Por qué construir un videojuego?

El desarrollo de videojuegos combina múltiples disciplinas de ingeniería de software, incluyendo arquitectura, diseño de sistemas, renderizado y manejo de eventos.

## ¿Qué tecnologías utiliza?

Principalmente Python y Pygame.

## ¿Es un motor de videojuegos?

No. Es un videojuego completo que implementa varios conceptos presentes en motores ligeros.

## ¿Es Open Source?

Sí. Todo el código fuente está disponible públicamente en GitHub.

## ¿A quién está dirigido?

A desarrolladores interesados en Python, videojuegos, arquitectura de software y programación orientada a objetos.
