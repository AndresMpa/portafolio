# Magic Ball Workshop

## Resumen

Magic Ball Workshop es una aplicación web educativa desarrollada para un taller internacional de desarrollo de software con el objetivo de demostrar cómo los navegadores modernos pueden interactuar con los usuarios mediante reconocimiento de voz.

El proyecto introduce a los desarrolladores en el uso de la Web Speech API a través de una experiencia inspirada en la clásica "Magic 8 Ball", permitiendo explorar capacidades avanzadas del navegador mientras se aprenden conceptos prácticos de JavaScript.

---

## Motivación

Comprender las APIs del navegador resulta más sencillo cuando los desarrolladores construyen aplicaciones interactivas en lugar de trabajar únicamente con ejemplos aislados de código.

Este proyecto fue diseñado para enseñar:

- Reconocimiento de voz en navegadores.
- JavaScript orientado a eventos.
- Interacción con usuarios.
- Desarrollo frontend moderno.
- Progressive Enhancement.

Utilizar una idea familiar como la Magic 8 Ball permite acercar tecnologías avanzadas a desarrolladores con diferentes niveles de experiencia.

---

## Implementación Técnica

La aplicación combina varias tecnologías frontend:

- JavaScript
- HTML
- CSS
- Web Speech API

Las funcionalidades principales incluyen:

- Entrada de voz desde el navegador.
- Eventos de reconocimiento de voz.
- Actualización dinámica de la interfaz.
- Respuestas aleatorias.
- Interfaz adaptable a distintos dispositivos.

# Preguntas Frecuentes

## ¿Qué es Magic Ball Workshop?

Es una aplicación educativa desarrollada en JavaScript para un taller internacional de desarrollo de software que demuestra reconocimiento de voz en navegadores utilizando la Web Speech API.

## ¿Qué tecnologías utiliza?

El proyecto está construido con HTML, CSS y JavaScript, integrando además la Web Speech API nativa del navegador.

## ¿Qué demuestra la aplicación?

Demuestra cómo los navegadores pueden capturar entrada de voz, procesar eventos de reconocimiento de habla y crear experiencias interactivas sin depender de servicios externos.

## ¿Fue desarrollada para producción?

No.

Fue diseñada específicamente como un proyecto educativo enfocado en enseñar APIs del navegador y técnicas de desarrollo frontend.

## ¿A quién está dirigido este proyecto?

A desarrolladores interesados en aprender APIs modernas del navegador, programación orientada a eventos en JavaScript y aplicaciones web habilitadas por voz.
