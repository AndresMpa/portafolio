# Advice Generator App

## Descripción General

Advice Generator App es una aplicación web frontend que muestra mensajes de consejos generados aleatoriamente a través de una interfaz simple y adaptable.

El proyecto fue creado como parte de mi proceso de aprendizaje en desarrollo frontend y se enfoca en fortalecer habilidades fundamentales de desarrollo web como programación en JavaScript, manipulación del DOM, diseño responsivo y arquitectura CSS utilizando Less.

Aunque se trata de una aplicación intencionalmente sencilla, proporcionó experiencia práctica valiosa en técnicas modernas de desarrollo frontend.

---

## Motivación

Una de las formas más efectivas de aprender desarrollo frontend es construir aplicaciones completas desde cero.

Este proyecto fue desarrollado para explorar:

- Fundamentos de JavaScript
- Interfaces de usuario dinámicas
- Manipulación del DOM
- Diseños responsivos
- Preprocesamiento CSS con Less
- Generación de contenido basada en APIs

El objetivo no era únicamente construir una aplicación funcional, sino también comprender mejor cómo las aplicaciones frontend gestionan datos e interacciones con los usuarios.

---

## Funcionalidades

La aplicación proporciona:

- Generación aleatoria de consejos
- Controles interactivos para el usuario
- Interfaz adaptable a diferentes dispositivos
- Diseño visual moderno
- Renderizado dinámico de contenido

El proyecto prioriza la simplicidad manteniendo una experiencia de usuario cuidada.
